# CLI Application

## Commands

- List contacts: `node index.js --action list`
- Get contact by ID: `node index.js --action get --id <contactId>`
- Add contact: `node index.js --action add --name <name> --email <email> --phone <phone>`
- Remove contact: `node index.js --action remove --id <contactId>`
